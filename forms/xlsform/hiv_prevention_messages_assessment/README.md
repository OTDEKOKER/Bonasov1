# HIV Prevention Messages Assessment (XLSForm)

This folder contains a starter **XLSForm** exported as `survey.csv` + `choices.csv`.

## Import (Kobo/ODK)
1. Create a new Excel workbook.
2. Create two sheets named `survey` and `choices`.
3. Copy/paste the contents of `survey.csv` into the `survey` sheet.
4. Copy/paste the contents of `choices.csv` into the `choices` sheet.
5. Save as `.xlsx` and upload to KoboToolbox/ODK.

## Notes
- This is a **v1 starter** based on the questions provided in chat (some question numbers were missing).
- You will likely want to add/adjust skip logic, required flags, and location choices.
